WP Tweetbox adds a highly customizable Tweetbox at the end of blog posts and pages. Tweets are branded with your own website URL.

== Description ==

WP Tweetbox adds a manual tweetbox with Twitter-style editor at the end of posts and pages.

**Features:**

* Tweets branded to your website
* Fully Customizable Tweetbox
* Tweet text can be auto-generated
* You can specify a custom Tweet text for individual posts/pages
* Tweetbox can be disabled on individual posts/pages
* Supports Su.pr, Bit.ly, Bitly.pro, Wp.me URL shorteners
* You can use a WordPress short URL with post ID or long permalink URL as well

== Installation ==
Upload the WP Tweetbox plugin folder to your plugins directory, activate it and configure the options on Settings page. Detailed instructions follow:

1. Extract and upload the 'wp-tweetbox' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to WP Tweetbox Settings page and adjust the options as described there.
4. Review and Save Changes.
5. WP Tweetbox box should now appear on your blog posts.
